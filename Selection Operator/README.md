## Вопрос
Какого типа может быть переменная в `switch`? Приведите примеры для всех возможных случаев.
## Ответ

Примитивные типы:
- `byte`
- `short`
- `char`
- `int`

Обертки для примитивов:
- `Byte`
- `Short`
- `Character`
- `Integer`

Строки:
- `String`

Перечисления (enums):
- `Enum`-тип (например, перечисление).
